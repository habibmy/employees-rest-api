Get Started

This Rest-Api gives json responses about the Employee.

Routes

Avialable methods

Supported Method Endpoint Resource
GET 
/api
 		Retrieve all employees from the database.
POST 
/api
 		Create new Employee. Send Details in Body with Json Format.
Example: 
  {
  "name": "Habib",
  "age": 25,
  "gender": "male",
  "designation": "Field Executive",
  "department": "Marketing"
  }
GET 
/api/:employeeId
 		Retrieve single employee from the database.
PUT 
/api/:employeeId
 		Update single employee detail in the database. Send the fields to be updated in Body with json formating.
Example: 
  {
  "age": 32
  }
DELETE 
/api/:employeeId
 		Delete a employee with the specified employeeId in the request
GET 
/api/department/:department
 		Find employee with a department.
